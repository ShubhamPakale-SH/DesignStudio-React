================================================================
           TOP 30 JAVA KEYWORDS - NOTES
================================================================

 #   KEYWORD         DESCRIPTION
----------------------------------------------------------------
 1.  class           Defines a class (blueprint for objects).
 2.  interface       Declares an interface (contract for classes to implement).
 3.  extends         Used for class inheritance (child extends parent).
 4.  implements      Used when a class implements an interface.
 5.  abstract        Declares an abstract class or method (no body, must be overridden).
 6.  new             Creates a new object/instance of a class.
 7.  this            Refers to the current instance of the class.
 8.  super           Refers to the parent class constructor or method.
 9.  static          Belongs to the class itself, not to any instance.
10.  final           Makes a variable constant, a method non-overridable, or a class non-inheritable.
11.  void            Specifies that a method does not return any value.
12.  return          Exits a method and optionally returns a value.
13.  if              Executes a block of code if a condition is true.
14.  else            Executes a block of code if the if condition is false.
15.  switch          Selects one of many code blocks to execute based on a value.
16.  case            Defines a branch in a switch statement.
17.  for             Looping construct that runs a block a set number of times.
18.  while           Looping construct that runs while a condition is true.
19.  do              Executes a block at least once, then repeats while condition is true.
20.  break           Exits a loop or switch statement immediately.
21.  continue        Skips the current loop iteration and moves to the next.
22.  try             Defines a block of code to test for exceptions.
23.  catch           Handles exceptions thrown in the try block.
24.  finally         Block that always executes after try-catch, used for cleanup.
25.  throw           Manually throws an exception.
26.  throws          Declares that a method may throw one or more exceptions.
27.  public          Access modifier — accessible from anywhere.
28.  private         Access modifier — accessible only within the same class.
29.  protected       Access modifier — accessible within the package and subclasses.
30.  instanceof      Checks whether an object is an instance of a specific class.
